# 보안 관련 중요 사항

## ⚠️ 중요한 보안 주의사항

### 1. GitHub 토큰 관리
- **절대 GitHub Personal Access Token을 코드에 직접 입력하지 마세요**
- `config.js` 파일은 `.gitignore`에 포함되어 GitHub에 푸시되지 않습니다
- 실제 운영시에는 `config.template.js`를 `config.js`로 복사하여 사용하세요

### 2. 관리자 인증 정보
- 관리자 이메일과 비밀번호가 코드에 포함되어 있습니다
- 실제 운영시에는 다음 조치를 취하세요:
  - 비밀번호를 해시화하여 저장
  - 환경변수 사용
  - 더 안전한 인증 방식 도입

### 3. 파일 구조
```
config.js           (실제 설정 - Git에서 제외됨)
config.template.js  (템플릿 - 안전하게 공유 가능)
.gitignore         (민감한 파일들을 Git에서 제외)
```

### 4. 배포 전 체크리스트
- [ ] `config.js`에서 실제 토큰 값 제거
- [ ] 개발용 설정들 제거
- [ ] `.gitignore`가 제대로 작동하는지 확인
- [ ] 민감한 정보가 커밋에 포함되지 않았는지 확인

### 5. 권장 운영 방식
1. 로컬에서만 `config.js` 사용
2. 서버 배포시 환경변수 사용
3. GitHub Actions 등에서는 Secrets 사용

## 현재 임시 설정
현재는 개발 편의를 위해 토큰이 포함되어 있지만, **실제 운영시에는 반드시 제거해야 합니다**.
